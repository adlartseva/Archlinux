quick start:
extract .conky .Conky to your home folder
extract fonts to ~/.local/share
extract the launch_conky.sh file to anywhere you wish; make it executable (i.e.) chmod u+x launch_conky.sh
configure variables to your liking, then run the script. Adjust as needed.
